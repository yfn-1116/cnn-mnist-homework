# data 目录说明

本目录负责数据集读取、预处理和 `DataLoader` 构建。

## 文件说明

- `loader.py`：构建训练集、验证集和测试集的 `DataLoader`
- `transforms.py`：定义训练和测试阶段的数据预处理与增强流程

## 当前数据流程

项目使用 `torchvision.datasets.MNIST` 读取数据，并在需要时自动下载。  
训练集会被拆分为：

- 训练集
- 验证集

测试集单独保留用于最终评估。

## 预处理逻辑

默认预处理包括：

- `ToTensor()`
- 按 MNIST 统计量进行归一化

当配置中启用 `augmentation` 时，训练集还会加入轻量数据增强，例如：

- 随机旋转
- 随机平移

测试集和验证集不使用数据增强，只做基础预处理。
